# 🚀 Project 1: Smart Agriculture Platform

![Screenshot](../assets/projects/project1.png)

## Overview
This project consists of a **smart agriculture monitoring system** integrating IoT sensors with a cloud backend.  
It helps farmers measure **pH, conductivity, and environmental variables**.

## Tech Stack
- Backend: Spring Boot (AWS Elastic Beanstalk)
- Frontend: Angular (Amplify Hosting)
- Database: SQL Server
- Infrastructure: Docker, Nginx, AWS S3

## Highlights
- 🌱 Real-time sensor monitoring  
- 📊 Interactive dashboards with charts  
- 🔐 Secure authentication with JWT  

## Role
- Designed and implemented the backend APIs  
- Deployed cloud infrastructure in AWS  
- Integrated IoT device communication
